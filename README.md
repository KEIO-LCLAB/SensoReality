# SensorReality
SensorReality, the use of virtual sesnors under the XR context as an augmentation tool for the design of interactive system.

## Environment
* win 10/11
* unity2022.3.32+
* Meta Quest All-in-one 65+
  
## Requirements
* Meta Quest 2/3/Pro

## Features
### supported sensors
* IMU Sensor
* Distance Sensor
* Light intensity Sensor (W.I.P)
